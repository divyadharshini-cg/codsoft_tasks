# 🎮 Tic-Tac-Toe AI

A modern, feature-rich Tic-Tac-Toe game developed in Python with an unbeatable AI powered by the **Minimax Algorithm** and **Alpha-Beta Pruning**.

This project was developed as **Task 2** for the **CodeSoft Artificial Intelligence Internship**.

---

## 📌 Features

- 🤖 Unbeatable AI using Minimax Algorithm
- ⚡ Alpha-Beta Pruning for optimized decision making
- 🎨 Modern Dark Theme UI
- 🎮 Play against AI or another player
- 📊 Game Statistics
- ⚙️ Settings Panel
- 🏆 Win, Lose & Draw Detection
- ✨ Smooth animations
- 🔊 Sound effects (if enabled)

---

## 🧠 AI Algorithm

The AI uses the **Minimax Algorithm**, a game theory algorithm that evaluates every possible move and always selects the optimal one.

### Difficulty Levels

- Easy → Random moves
- Medium → Mix of Random & Minimax
- Hard → Pure Minimax
- Expert → Minimax + Alpha-Beta Pruning

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/codsoft_tasks.git
```

Go to the project folder:

```bash
cd Task2_TicTacToe_AI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
